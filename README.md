# Advent of Code

Exercise info: [Syntax Scoring](https://adventofcode.com/2021/day/10)

## Day 10 Exercise

I completed this exercise in Typescript using TDD with the help of Jest.

You can install and run the script to validate it's functionality.

### Install

```bash
yarn
```

### Run unit tests

```bash
yarn test
```

### Run Part 1

```bash
yarn part1
```

### Run Part 2

```bash
yarn part2
```
